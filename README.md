# SQL Query Simulator

A web-based SQL simulator that allows users to execute basic SQL-like queries on a sample dataset.

## Features
- SELECT * query simulation
- WHERE clause filtering
- ORDER BY sorting (ASC/DESC)
- INSERT query simulation
- Interactive and colorful UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Used
- Relational Database Management Systems
- Query Parsing
- Data Filtering and Sorting
